# AGI Agent 核心循环迭代方案

> 本文档描述 AGI Agent 核心执行循环的智能化迭代改进方案，旨在实现真正的"持续学习智能体"。

---

## 1. 背景与目标

### 1.1 现有核心循环

AGI Agent 当前基于 **ReAct 模型** 执行任务，核心循环为：

```
Plan → Act → Observe → Reflect
```

默认 50 轮迭代，支持多轮工具调用与自我反思。

### 1.2 存在的问题

| 问题 | 描述 |
|------|------|
| **记忆沉睡** | 长期记忆系统存在但 Agent 不会主动检索 |
| **经验孤岛** | Skills 经验系统存在但未在执行循环中自动使用 |
| **缺乏学习** | 任务完成后不自动积累经验 |
| **效率低下** | 每次任务从零开始，不借鉴历史 |

### 1.3 改进目标

让 Agent 在每次任务中都能：
- **做之前想一想** → 主动检索记忆 + 匹配经验
- **做之中学一学** → 借鉴成功模式
- **做之后记一记** → 自动存档经验
- **定期清一清** → 合并优化经验库

| 属性 | 描述 |
|------|------|
| **位置** | `src/mem/` |

### 3.2 各阶段详细说明

#### 阶段0：智能准备

**目标**：在执行任务前自动检索相关记忆和经验

**0.1 自主记忆检索**

```python
async def auto_memory_check(requirement: str) -> Dict:
    """
    智能记忆检索入口
    
    触发时机：
    - 任务开始前
    - 遇到困难时
    - 需要借鉴历史时
    """
    # 并行检索两种记忆
    results = await asyncio.gather(
        # 原始记忆（具体细节）
        read_memory_auto(f"与'{requirement}'相关的任务"),
        # 总结记忆（模式/趋势）
        search_memoir_memories_by_query(requirement, top_k=3)
    )
    
    preliminary, memoir = results
    
    return {
        "specific_cases": preliminary['results'],  # 具体案例
        "patterns": memoir['results'],              # 成功模式
        "task_type": classify(requirement)         # 任务分类
    }
```

**0.2 自主经验匹配**

```python
class SkillMatcher:
    """
    经验匹配器 - 核心智能组件
    """
    
    def match_best_skill(self, requirement: str, context: Dict) -> Optional[Skill]:
        """
        找到最匹配的经验
        
        核心机制：基于 TF-IDF + cosine_similarity 的智能匹配
        """
        # 1. 构建查询文本
        query = f"{requirement} {context.get('task_type', '')}"
        
        # 2. 加载所有skill
        skills = self._load_all_skills()
        
        # 3. TF-IDF向量化
        texts = [skill.content + skill.front_matter.usage_conditions 
                 for skill in skills]
        tfidf_matrix = self.vectorizer.fit_transform([query] + texts)
        
        # 4. 计算相似度
        similarities = cosine_similarity(tfidf_matrix[0], tfidf_matrix[1:])[0]
        
        # 5. 选择最佳匹配
        best_idx = similarities.argmax()
        best_skill = skills[best_idx]
        
        # 6. 检查质量阈值 (quality_index >= 0.5)
        if best_skill.quality_index >= self.min_quality_threshold:
            best_skill.fetch_count += 1  # 更新使用次数
            return best_skill
        
        return None
```

#### 阶段1-4：执行循环

与现有 ReAct 循环保持一致，但使用增强后的上下文。

```
while (未达到最大轮数) and (任务未完成):
    # Plan: 使用增强后的上下文
    current_prompt = enhanced_context + base_prompt
    
    # Act: 调用ToolExecutor执行
    result = self.executor.execute_subtask(current_prompt, ...)
    
    # Observe: 获取工具执行结果
    task_history.append(result)
    
    # Reflect: 检查是否完成
    if task_completed:
        break
    task_round += 1
```

#### 阶段5：经验存档

**目标**：任务成功后自动积累经验

```python
class ExperienceArchiver:
    """
    经验存档器 - 将成功任务转化为可复用经验
    """
    
    def archive_success(self, task_result: Dict, requirement: str) -> bool:
        """
        存档成功任务
        """
        # 1. 提取关键模式
        patterns = self._extract_patterns(task_result)
        
        # 2. 创建Skill文件
        skill_data = {
            'skill_id': str(int(time.time())),
            'title': self._generate_title(requirement),
            'usage_conditions': self._extract_conditions(requirement, patterns),
            'quality_index': self._calculate_quality(task_result),
            'fetch_count': 0,
            'related_code': self._find_related_files(task_result),
            'task_directories': [task_result['workspace_dir']],
            'created_at': datetime.now().isoformat(),
        }
        
        # 3. 保存到experience目录
        return self._save_skill(skill_data, task_result)
```

### 3.3 经验自动进化

**SkillManager 定期任务**：

```python
class SkillManager:
    """
    经验进化管理器
    """
    
    def evolve(self):
        """
        经验自我优化流程
        """
        # Step 1: 基于TF-IDF合并相似经验 (相似度 > 0.7)
        self._merge_similar_skills(threshold=0.7)
        
        # Step 2: DBSCAN聚类
        clusters = self._cluster_skills_dbscan()
        
        # Step 3: LLM决策是否整合
        for cluster in clusters:
            decision = self._llm_merge_decision(cluster)
            if decision.should_merge:
                self._integrate_skills(cluster, decision)
        
        # Step 4: 清理长期未使用的经验 (30天无使用)
        self._clean_unused_skills(days=30)
```

## 7. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| 记忆检索引入延迟 | 执行速度下降 | 异步并行执行，设置超时 |
| 经验匹配误判 | 执行策略错误 | 设置质量阈值，仅使用高质量经验 |
| 经验库膨胀 | 存储占用增加 | 定期清理 + 自动合并 |
| LLM整合决策失败 | 经验质量下降 | 降级为纯规则合并 |
