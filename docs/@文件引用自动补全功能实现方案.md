## 一、需求概述

### 1.1 功能描述

在 AGI-Agent 前端聊天界面的消息发送模态中，实现类似 GitHub/VS Code 的 `@` 符号文件引用自动补全功能。用户输入 `@` 后，自动弹出下拉菜单，以**扁平化目录树**方式显示工作区文件，支持模糊搜索和键盘导航选择。

### 1.2 用户体验目标

```
┌─────────────────────────────────────────────────────────────┐
│ 消息输入框                                                   │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ 请帮我分析 @ ← 光标在此位置触发自动补全                 │ │
│ └─────────────────────────────────────────────────────────┘ │
│            ↓ 弹出下拉菜单（带层级线条的树形结构）               │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │ 📁 docs/                                               │ │
│ │ ├─ 📄 plan.md                                          │ │
│ │ ├─ 📄 readme.md                                        │ │
│ │ └─ 📁 src/                                             │ │
│ │   ├─ 📄 main.py                                        │ │
│ │   └─ 📁 utils/                                         │ │
│ │     └─ 📄 helper.py                                    │ │
│ └─ 📁 assets/                                            │ │
│   └─ 📄 logo.png                                         │ │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 预期效果

- 用户输入 `@` 时立即弹出**扁平化目录列表**
- 文件夹和文件**统一列表显示**，无需分组切换
- 支持直接引用**文件夹**，不仅仅是文件
- 继续输入字符时实时过滤匹配的文件/文件夹
- 支持键盘导航（↑↓ Enter Esc Tab）
- 选择后插入 `@文件路径` 或 `@文件夹路径/` 到输入框
- 自动存储路径映射到 `window.attachedFiles`

### 1.4 核心改进点（v3.0）

| 改进点 | v2.0 行为 | v3.0 行为 |
|--------|-----------|----------|
| 显示方式 | 分组目录树显示（文件夹/文件分组） | 带层级线条的树形结构 |
| 根目录显示 | 显示 `output_xxx/` 根目录 | ❌ 不显示根目录本身 |
| 路径显示 | 显示父目录路径 | ✅ 层级线条 + 缩进 |
| 列表项 | 左侧图标 + 名称 + 路径（右侧） | 图标 + 线条 + 名称 |
| 排序逻辑 | 按深度排序 | ✅ 保持遍历顺序（树形结构） |

### 1.5 问题修复记录（v3.0）

| 问题 | 原因 | 修复方案 |
|------|------|----------|
| 索引到其他工作目录的文件 | `buildFileCache` 遍历了所有 `cachedDirectoryTree` | ✅ 只索引 `selectedDirectory` 对应的目录 |
| 切换目录后缓存不更新 | `selectDirectory` 没有刷新 @ 引用缓存 | ✅ 切换目录时调用 `refreshMentionCache()` |

#### 目录匹配优先级

```
1. selectedDirectory (用户当前选中的目录)  ← 最高优先级
2. is_current (后端标识的当前目录)          ← 备选
3. 第一个目录                               ← 兜底
```

### 3.1 方案选型

| 方案 | 优点 | 缺点 | 推荐 |
|------|------|------|------|
| A: 基于现有目录树过滤 | 无需后端改动、性能好 | 需要缓存目录数据 | ✅ **推荐** |

**选择方案 A**，因为：
1. 现有 `/api/output-dirs` 已返回完整目录树
2. 只需前端过滤逻辑
3. 性能足够（本地数据过滤）

#### 5.3.3 输入事件处理

```javascript
// 处理输入事件
function handleMentionInput(e) {
    const input = e.target || e.currentTarget;
    const cursorPos = input.selectionStart;
    const textBeforeCursor = input.value.substring(0, cursorPos);
    
    // 检测 @ 触发器
    // 匹配 @ 后面没有空格和 @ 的文本
    const atMatch = textBeforeCursor.match(/@([^\s@]*)$/);
    
    if (atMatch) {
        // 找到了 @ 触发器
        const query = atMatch[1];
        const triggerPos = cursorPos - query.length - 1; // @ 符号的位置
        
        mentionState.query = query;
        mentionState.triggerPosition = triggerPos;
        mentionState.cursorPosition = cursorPos;
        
        // 过滤文件列表
        const filteredItems = filterFilesForMention(query);
        mentionState.items = filteredItems;
        mentionState.selectedIndex = 0;
        
        // 显示下拉菜单
        showMentionDropdown(filteredItems);
    } else {
        // 没有 @ 触发器，隐藏下拉菜单
        hideMentionDropdown();
    }
}
```

#### 5.3.4 文件过滤逻辑

```javascript
// 过滤文件列表
function filterFilesForMention(query) {
    const queryLower = query.toLowerCase();
    const results = [];
    
    // 优先从缓存中获取文件列表
    if (cachedFileList.length === 0) {
        buildFileCache();
    }
    
    // 遍历缓存的文件列表
    for (const file of cachedFileList) {
        const nameLower = file.name.toLowerCase();
        const pathLower = (file.path || '').toLowerCase();
        
        // 匹配文件名或路径
        if (nameLower.includes(queryLower) || pathLower.includes(queryLower)) {
            results.push({
                name: file.name,
                path: file.path,
                type: file.type,
                displayName: file.path ? `@${file.path}` : `@${file.name}`,
                score: calculateMatchScore(file.name, query)
            });
        }
        
        // 限制结果数量
        if (results.length >= 15) break;
    }
    
    // 按匹配分数排序
    results.sort((a, b) => b.score - a.score);
    
    return results;
}

// 构建文件缓存
function buildFileCache() {
    cachedFileList = [];
    
    // 从 window.cachedDirectoryTree 递归收集文件
    function traverse(files) {
        if (!files) return;
        for (const file of files) {
            if (file.type === 'file') {
                cachedFileList.push({
                    name: file.name,
                    path: file.path || file.relativePath || file.name,
                    type: 'file'
                });
            }
            if (file.children && file.children.length > 0) {
                traverse(file.children);
            }
        }
    }
    
    // 尝试从页面数据获取目录树
    const directoryData = window.cachedDirectoryTree;
    if (directoryData && Array.isArray(directoryData)) {
        traverse(directoryData);
    } else {
        // 备选：从 DOM 收集
        const fileItems = document.querySelectorAll('.file-item:not(.directory)');
        fileItems.forEach(item => {
            const name = item.querySelector('.file-name')?.textContent || 
                        item.textContent;
            cachedFileList.push({
                name: name.trim(),
                path: name.trim(),
                type: 'file'
            });
        });
    }
    
    console.log(`[Mention] 文件缓存已构建，共 ${cachedFileList.length} 个文件`);
}

// 计算匹配分数（用于排序）
function calculateMatchScore(fileName, query) {
    const nameLower = fileName.toLowerCase();
    const queryLower = query.toLowerCase();
    
    // 精确前缀匹配得分最高
    if (nameLower.startsWith(queryLower)) return 100;
    
    // 包含匹配
    if (nameLower.includes(queryLower)) return 50;
    
    return 10;
}

// 刷新文件缓存（在目录刷新时调用）
function refreshMentionCache() {
    cachedFileList = [];
    buildFileCache();
}
```

```javascript
function showMentionDropdown(items) {
    if (!mentionDropdown || !mentionList) return;

            li.className = 'mention-item' + (index === mentionState.selectedIndex ? ' selected' : '');
            li.dataset.index = index;

                ${item.path !== item.name ? `<span class="mention-item-path">${escapeHtml(item.path)}</span>` : ''}
            `;
            
            // 点击选择
            li.addEventListener('click', () => selectMentionItem(index));
            
            mentionList.appendChild(li);
        });
    }
    
    // 定位下拉菜单
    positionMentionDropdown();
    
    // 显示
    mentionDropdown.style.display = 'block';
    mentionState.visible = true;
    
    // 滚动到选中项
    scrollToSelectedMentionItem();
}

// 隐藏下拉菜单
function hideMentionDropdown() {
    if (!mentionDropdown) return;
    
    mentionDropdown.style.display = 'none';
    mentionState.visible = false;
    mentionState.items = [];
    mentionState.selectedIndex = 0;
}

// 定位下拉菜单
function positionMentionDropdown() {
    if (!mentionDropdown || !userInputElement) return;
    
    const inputRect = userInputElement.getBoundingClientRect();
    const viewportHeight = window.innerHeight;
    
    // 优先显示在输入框下方
    let top = inputRect.bottom + window.scrollY + 4;
    
    // 如果下方空间不够，显示在上方
    const dropdownHeight = 240;
    if (top + dropdownHeight > viewportHeight + window.scrollY) {
        top = inputRect.top + window.scrollY - dropdownHeight - 4;
    }
    
    // 设置位置
    mentionDropdown.style.top = `${top}px`;
    mentionDropdown.style.left = `${inputRect.left + window.scrollX}px`;
    mentionDropdown.style.width = `${Math.max(inputRect.width, 280)}px`;
}

// 滚动到选中项
function scrollToSelectedMentionItem() {
    const selected = mentionList.querySelector('.mention-item.selected');
    if (selected) {
        selected.scrollIntoView({ block: 'nearest', behavior: 'smooth' });
    }
}
```

#### 5.3.6 键盘导航处理

```javascript
// 处理键盘事件
function handleMentionKeydown(e) {
    if (!mentionState.visible || mentionState.items.length === 0) {
        return;
    }
    
    switch (e.key) {
        case 'ArrowDown':
            e.preventDefault();
            e.stopPropagation();
            mentionState.selectedIndex = Math.min(
                mentionState.selectedIndex + 1,
                mentionState.items.length - 1
            );
            updateMentionSelection();
            break;
            
        case 'ArrowUp':
            e.preventDefault();
            e.stopPropagation();
            mentionState.selectedIndex = Math.max(
                mentionState.selectedIndex - 1,
                0
            );
            updateMentionSelection();
            break;
            
        case 'Enter':
            e.preventDefault();
            e.stopPropagation();
            selectMentionItem(mentionState.selectedIndex);
            break;
            
        case 'Escape':
            e.preventDefault();
            e.stopPropagation();
            hideMentionDropdown();
            break;
            
        case 'Tab':
```

```javascript
// 选择文件项
function selectMentionItem(index) {

// 替换 @ 触发器文本
function replaceMentionText(newText) {
    const input = userInputElement;
    const value = input.value;
    
    // 获取 @ 符号的位置和范围
    const triggerStart = mentionState.triggerPosition;
    const triggerEnd = mentionState.cursorPosition;
    
    // 保留 @ 前的空格（如果有）
    const textBefore = value.substring(0, triggerStart);
    const needsSpaceBefore = textBefore.length > 0 && 
                             !textBefore.endsWith(' ') && 
                             !textBefore.endsWith('\n');
    
    // 保留触发器后的空格（如果有）
    const textAfter = value.substring(triggerEnd);
    const needsSpaceAfter = textAfter.length > 0 && 
                             !textAfter.startsWith(' ') && 
                             !textAfter.startsWith('\n');
    
    // 构造新文本
    const spaceBefore = needsSpaceBefore ? ' ' : '';
    const spaceAfter = needsSpaceAfter ? ' ' : '';
    const replacement = spaceBefore + newText + spaceAfter;
    
    // 替换
    input.value = value.substring(0, triggerStart) + replacement + textAfter;
    
    // 设置光标位置（在新文本之后）
    const newCursorPos = triggerStart + replacement.length;
    input.setSelectionRange(newCursorPos, newCursorPos);
    
    // 聚焦输入框
    input.focus();
}
```

```javascript
// 核心逻辑：深度优先遍历，只索引当前选中目录
function buildFileCache() {
    cachedFileList = [];

    function traverse(dirEntry, parentPath) {
        var files = dirEntry.files || dirEntry.children;
        for (var file of files) {
            var fullPath = parentPath ? parentPath + '/' + file.name : file.name;
            cachedFileList.push({
                name: file.name,
                path: fullPath,
                type: file.type,
                displayName: '@' + fullPath + (file.type === 'directory' ? '/' : '')
            });
            // 递归处理子目录
            if (file.children && file.children.length > 0) {
                traverse(file, fullPath);
            }
        }
    }

    // 只索引 selectedDirectory 对应的目录
    // 优先级：selectedDirectory > is_current > 第一个目录
    // ...
}
```

#### 2. filterFilesForMention() - 过滤与排序

```javascript
// 核心逻辑：保持遍历顺序，不打乱树形结构
function filterFilesForMention(query) {
    var results = [];
    for (var file of cachedFileList) {
        if (匹配条件) {
            results.push({
                name: file.name,
                path: file.path,
                type: file.type,
                displayName: file.displayName,
                score: calculateMatchScore(file.name, query),
                depth: file.path.split('/').length - 1
            });
        }
        if (results.length >= 50) break;
    }
    // 保持原有遍历顺序，不重新排序
    return { items: results, totalCount: results.length };
}
```

#### 3. createMentionItem() - 创建列表项（带层级线条）

```javascript
function createMentionItem(item, index) {
    var li = document.createElement('li');
    li.className = 'mention-item' + (index === mentionState.selectedIndex ? ' selected' : '');
    li.dataset.index = index;

    var isFolder = item.type === 'directory';
    var iconClass = getFileIconClass(item.name);
    
    // 文件夹图标特殊处理：folder 类放在外层 span，图标本身用 fas fa-folder
    var iconSpanClass = 'mention-item-icon';
    if (isFolder) {
        iconSpanClass += ' folder';
        iconClass = 'fas fa-folder';
    }

    // 计算缩进深度
    var pathParts = item.path.split('/');
    var depth = Math.max(0, pathParts.length - 1);

    // 构建层级线条（│ 垂直线 + ├─ 分支线）
    var treeLines = '';
    if (depth > 0) {
        for (var d = 0; d < depth - 1; d++) {
            treeLines += '<span class="tree-vert-line">│ </span>';
        }
        treeLines += '<span class="tree-branch">├─</span>';
    }

    li.innerHTML = '<span class="' + iconSpanClass + '"><i class="' + iconClass + '"></i></span>' +
        '<div class="mention-item-content">' +
        '<span class="tree-lines">' + treeLines + '</span>' +
        '<span class="mention-item-name">' + escapeHtml(item.name) + (isFolder ? '/' : '') + '</span>' +
        '</div>';
    
    // 点击选择
    li.addEventListener('click', (function(idx) {
        return function() {
            selectMentionItem(idx);
        };
    })(index));

    return li;
}
```
